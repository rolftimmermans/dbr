def_class(superklass, methodname)

Define unnamed class which has methodname as instance method, and return it.

example: class MyClass_ def initialize(arg1, arg2) @arg1 = arg1; @arg2 = arg2 end end filename = ‘example.rhtml’ # @arg1 and @arg2 are used in example.rhtml erb = ERB.new(File.read(filename)) erb.filename = filename MyClass = erb.def_class(MyClass_, ‘render()’) print MyClass.new(‘foo’, 123).render()

def_method(mod, methodname, fname)

Define methodname as instance method of mod from compiled ruby source.

example: filename = ‘example.rhtml’ # ‘arg1’ and ‘arg2’ are used in example.rhtml erb = ERB.new(File.read(filename)) erb.def_method(MyClass, ‘render(arg1, arg2)’, filename) print MyClass.new.render(‘foo’, 123)

def_module(methodname)

Create unnamed module, define methodname as instance method of it, and return it.

example: filename = ‘example.rhtml’ # ‘arg1’ and ‘arg2’ are used in example.rhtml erb = ERB.new(File.read(filename)) erb.filename = filename MyModule = erb.def_module(‘render(arg1, arg2)’) class MyClass include MyModule end

result(b)

Executes the generated ERB code to produce a completed template, returning the results of that code. (See ERB#new for details on how this process can be affected by safe_level.)

b accepts a Binding or Proc object which is used to set the context of code evaluation.
